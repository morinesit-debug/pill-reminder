# 반려동물 알약 알람 앱 (Pill Reminder)

반려동물의 알약 복용을 관리하고 알림을 제공하는 Flutter 앱입니다.

## 주요 기능

### 1. 알약 추가
- 알약 이름 입력
- 복용 주기 설정 (매일, 매주, 매월, 커스텀)
- 시작일 지정
- 알약 활성화/비활성화

### 2. 알약 복용 관리
- 오늘 복용해야 할 알약 목록 표시
- 복용 완료 체크
- 복용 건너뛰기 기능
- 실시간 복용 상태 업데이트

### 3. 캘린더 뷰
- 월별 캘린더로 복용 일정 확인
- 날짜별 복용 상태 시각화
- 특정 날짜 선택하여 상세 정보 확인

### 4. 알림 시스템
- 푸시 알림으로 복용 시간 알림
- 정확한 시간에 알림 발송
- 백그라운드에서도 알림 동작

### 5. 데이터 관리
- SQLite 데이터베이스로 데이터 저장
- 복용 기록 자동 생성
- 알약 정보 수정 및 삭제

## 기술 스택

- **Flutter**: 크로스 플랫폼 모바일 앱 개발
- **Provider**: 상태 관리
- **SQLite**: 로컬 데이터베이스
- **Flutter Local Notifications**: 로컬 알림
- **Table Calendar**: 캘린더 위젯
- **Timezone**: 시간대 관리

## 설치 및 실행

### 필수 요구사항
- Flutter SDK 3.8.0 이상
- Dart SDK
- iOS 12.0 이상 또는 Android 5.0 이상

### 설치 단계

1. 저장소 클론
```bash
git clone <repository-url>
cd pill_reminder
```

2. 의존성 설치
```bash
flutter pub get
```

3. 앱 실행
```bash
flutter run
```

## 사용법

### 알약 추가하기
1. 메인 화면의 '+' 버튼 탭
2. 알약 이름 입력
3. 복용 주기 선택 (매일/매주/매월/커스텀)
4. 시작일 선택
5. '추가하기' 버튼 탭

### 복용 체크하기
1. 오늘 탭에서 복용할 알약 확인
2. '복용 완료' 버튼으로 복용 체크
3. 또는 '건너뛰기' 버튼으로 건너뛰기

### 캘린더로 확인하기
1. 하단 네비게이션에서 '캘린더' 탭 선택
2. 원하는 날짜 선택
3. 해당 날짜의 복용 상태 확인
4. 필요시 복용 완료 또는 건너뛰기 처리

## 권한 설정

### Android
앱이 자동으로 필요한 권한을 요청합니다:
- 알림 권한
- 부팅 완료 수신 권한
- 진동 권한
- 정확한 알람 스케줄링 권한

### iOS
앱이 자동으로 필요한 권한을 요청합니다:
- 알림 권한
- 백그라운드 처리 권한

## 데이터 구조

### Pill (알약)
- id: 고유 식별자
- name: 알약 이름
- frequency: 복용 주기
- startDate: 시작일
- customDays: 커스텀 주기일 때 일수
- isActive: 활성화 여부
- createdAt: 생성일

### PillRecord (복용 기록)
- id: 고유 식별자
- pillId: 알약 ID
- scheduledDate: 예정된 복용일
- takenDate: 실제 복용일
- isTaken: 복용 완료 여부
- isSkipped: 건너뛰기 여부

## 알림 시스템

앱은 다음과 같은 경우에 알림을 발송합니다:
- 설정된 복용 시간에 도달했을 때
- 복용 주기에 따라 자동으로 스케줄링
- 다음 30일간의 알림을 미리 설정

## 문제 해결

### 알림이 오지 않는 경우
1. 앱 알림 권한 확인
2. 알약이 활성화되어 있는지 확인
3. 시작일이 올바르게 설정되었는지 확인

### 데이터가 사라진 경우
1. 앱 재시작
2. 데이터베이스 파일 확인
3. 필요시 앱 재설치
